2. Kopiere den **Preview Token** (NICHT Management Token!)
3. Füge in Vercel Environment Variables ein

## 🐛 Troubleshooting

### Problem: "Storyblok API not initialized"
**Lösung:** Überprüfe `VITE_STORYBLOK_TOKEN` in Vercel Environment Variables

### Problem: Google Script Timeout
**Lösung:** Script ist idempotent - einfach nochmal ausführen

### Problem: Inhalte werden nicht angezeigt
**Lösung:** 
1. Überprüfe Browser Console (F12)
2. Verifiziere Storyblok API Response
3. Stelle sicher, dass Stories published sind

**Mehr:** [STORYBLOK_MIGRATION_GUIDE.md → Troubleshooting](./STORYBLOK_MIGRATION_GUIDE.md#troubleshooting)
